# Rectal-Cancer-Dual-AI-Suite-cCR-LPLN-Predictor
Developed by Dr. Harishankar BGV, this dual-model AI predicts Complete Clinical Response (Watch &amp; Wait) and LPLN Sterilization in rectal cancer post-nCRT. Crucially, the AI proved that primary tumor anatomy (Age, Height, Thickness) dictates regional nodal radiosensitivity, outperforming traditional baseline nodal metrics.
